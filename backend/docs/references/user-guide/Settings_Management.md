# 설정 관리 가이드

## 개요
이 가이드는 설정 시스템의 사용 방법을 설명합니다.

## 설정 항목

### 추적 설정
| 설정 키 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| tracking.mode | STRING | "Auto" | 추적 모드 (Auto/Manual) |
| tracking.interval | INTEGER | 60 | 추적 주기 (초) |
| tracking.speed | INTEGER | 10 | 추적 속도 |
| tracking.accuracy | INTEGER | 1 | 추적 정확도 |

## 설정 관리 방법

### 1. 설정 조회
- 웹 인터페이스: 설정 메뉴에서 조회
- API: `GET /api/v1/settings` 호출

### 2. 설정 변경
1. 변경할 설정 선택
2. 새로운 값 입력
3. 변경 사유 입력 (선택사항)
4. 저장 버튼 클릭

### 3. 이력 조회
- 각 설정의 변경 이력 확인 가능
- 변경자, 변경 시각, 이전 값, 변경 사유 표시

## 캐시 관리
- 설정 변경 시 자동으로 캐시 업데이트
- 필요시 수동으로 캐시 새로고침 가능

## 모범 사례
1. 중요 설정 변경 시 사유 기록
2. 주기적인 설정 검토
3. 변경 이력 모니터링

## 문제 해결
- 설정이 적용되지 않을 경우
  1. 캐시 새로고침 시도
  2. 설정 값 형식 확인
  3. 권한 확인
