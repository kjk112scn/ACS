# Daily Logs - 일일 작업 로그

> 매일의 작업 내용을 기록하여 히스토리를 추적합니다.

## 목적

- 작업 히스토리 추적
- Why 기록 (왜 이 작업을 했는가)
- 진행 상황 가시화
- 팀 커뮤니케이션

## 파일 형식

`YYYY-MM-DD.md`

예시: `2026-01-06.md`

## 로그 내용

각 일일 로그에는 다음 내용이 포함됩니다:

1. **완료된 작업**
   - 작업명, 유형, 변경 요약
   - **Why**: 왜 이 작업을 했는가

2. **진행 중 작업**
   - 진행률, 다음 단계, 블로커

3. **커밋 목록**
   - 시간, 해시, 메시지

4. **내일 계획**
   - 예정된 작업

## 자동 생성

`/done` 스킬 실행 시 자동으로 일일 로그에 기록됩니다.
