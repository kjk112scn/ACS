## 이슈 발생 시

### 롤백 방법

```bash
# 특정 파일 롤백
git checkout HEAD~1 -- <파일경로>

# 전체 롤백
git revert HEAD
```

1. DevTools > Elements > Computed 탭에서 적용된 스타일 확인
2. `!important` 충돌 시 선택자 특이성 확인
3. Quasar 내부 스타일은 `:deep()` 필요
