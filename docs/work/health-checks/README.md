# Health Check Reports

프로젝트 건강 상태 일일 리포트 저장소

## 파일 명명 규칙

```
YYYY-MM-DD.md
```

예: `2026-01-20.md`

## 실행 방법

```
/health
```

## 리포트 항목

| 항목 | 설명 |
|------|------|
| 빌드 상태 | BE/FE 빌드 성공 여부 |
| 타입/린트 | TypeScript, ESLint 오류 |
| 기술 부채 | 대형 파일, TODO, console.log |
| 진행 중 작업 | active/ 폴더 현황 |
| 완료 작업 | 당일 완료된 작업 |

## 점수 기준

| 점수 | 등급 | 상태 |
|------|------|------|
| 90+ | A | 🟢 우수 |
| 80-89 | B | 🟢 양호 |
| 70-79 | C | 🟡 보통 |
| 60-69 | D | 🟠 주의 |
| 60 미만 | F | 🔴 위험 |
