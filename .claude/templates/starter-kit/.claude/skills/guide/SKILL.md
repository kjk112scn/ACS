# Guide - 사용법 안내

## 스킬 목록

| 스킬 | 용도 | 예시 |
|------|------|------|
| `/guide` | 이 안내 | `/guide` |
| `/plan` | 계획 수립 | `/plan 기능 추가` |
| `/done` | 작업 완료 + 커밋 | `/done` |
| `/health` | 빌드/품질 점검 | `/health` |

## 협업 대화 패턴

### 기본 원칙

1. **모든 작업 전 확인** - AI가 먼저 이해한 내용 확인
2. **양방향 의견 교환** - AI도 의견 제시, 사용자도 피드백
3. **단계별 진행** - Research → Plan → Implement

### 작업 시작 시 AI가 확인하는 것

```
사용자: "XXX 기능 추가해줘"

AI 응답 패턴:
┌─────────────────────────────────────────────────┐
│ 📋 요청 분석                                      │
│                                                 │
│ **이해한 내용:**                                  │
│ - XXX 기능 구현                                  │
│                                                 │
│ **확인 필요:**                                    │
│ 1. 주요 목표가 뭔가요?                            │
│ 2. 우선순위가 있나요?                             │
└─────────────────────────────────────────────────┘
```

### AI 의견 제시 패턴

```
┌─────────────────────────────────────────────────┐
│ 💡 내 의견                                       │
│                                                 │
│ **추천: {방안}**                                 │
│                                                 │
│ 이유:                                           │
│ 1. {이유1}                                      │
│ 2. {이유2}                                      │
│                                                 │
│ **대안들:**                                      │
│ | 방식 | 장점 | 단점 |                           │
│                                                 │
│ 어떻게 생각하세요?                               │
└─────────────────────────────────────────────────┘
```

### 단계별 작업 흐름

```
[1단계: 요청]
사용자 → 작업 요청

[2단계: 확인]  ← AI가 먼저 확인
AI → "이렇게 이해했는데 맞나요?"
   → "추가로 알아야 할 것: ..."

[3단계: 계획]
AI → 작업 계획 제시
   → "이 방향이 좋을 것 같은데, 어떻게 생각하세요?"

[4단계: 합의]
사용자 → 승인/수정 요청

[5단계: 실행]
AI → 구현 진행

[6단계: 완료]
AI → 결과 보고 + 다음 단계 안내
```

### 의견 충돌 시

```
사용자: "A 방식으로 하고 싶어"
AI: "A도 가능하지만, B가 더 나을 수 있어요.

     A 방식:
     + 장점
     - 단점

     B 방식:
     + 장점

     최종 결정은 사용자님께서 해주세요.
     A로 진행할까요?"
```
