# Status - 프로젝트 현황 보고 스킬

## 역할

CURRENT_STATUS.md를 기반으로 프로젝트 현황을 빠르게 파악하고 보고합니다.

**핵심 가치:**
- 세션 시작 시 빠른 컨텍스트 복구
- 진행 중/완료/예정 작업 한눈에 파악
- 다음 할 일 우선순위 제안

---

## 워크플로우

```
[1. 상태 로드] → [2. 분석] → [3. 보고]
       │             │           │
  CURRENT_STATUS  진행률 계산   요약 출력
  .md 읽기       우선순위 판단
```
