# Archive - 완료된 작업 보관소

> 완료된 작업 문서 아카이브

---

## 용도

- **재활용:** 유사한 버그/기능 작업 시 참고
- **히스토리:** 과거 결정 사항 추적
- **학습:** 이전 해결책 참조

---

## 폴더 구조

```
archive/
└── {YYYY-MM}_{Category}_{Name}/
    ├── README.md      # 개요 + 최종 상태
    ├── PROGRESS.md    # 완료된 체크리스트
    └── ...
```

---

## 아카이브 조건

| 조건 | 확인 |
|------|------|
| 진행률 100% | PROGRESS.md |
| 상태 "완료" | README.md |

---

## 부활 방법

```
/revive {키워드}

예: /revive Login
→ 관련 아카이브 검색 → active/로 복구
```

---

## 아카이브 목록

| 날짜 | 작업 | 유형 | 요약 |
|------|------|------|------|
| YYYY-MM | {Name} | Feature | {한줄요약} |
