## 문서 관리

### 업데이트 시점

| 상황 | 대응 |
|------|------|
| 새 도메인 개념 추가 | domain/ 에 문서 추가 |
| 아키텍처 변경 | architecture/ 업데이트 |
| 코드 구조 변경 | 관련 문서 갱신 |

### 문서 작성 가이드

1. **간결하게** - 핵심만 담기 (AI 컨텍스트 절약)
2. **예시 포함** - 코드 스니펫으로 이해도 향상
3. **링크 활용** - 상세 내용은 링크로 연결
